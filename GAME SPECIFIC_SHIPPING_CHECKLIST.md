# Shipping Checklist

## Code and repo
- Remove dead code
- Remove debug hotkeys unless intentionally kept
- Remove unused assets
- Confirm README is accurate
- Confirm install/run commands work from a clean clone
- Confirm `.gitignore` is sane
- Confirm asset licensing notes are documented

## Game quality
- Core loop is understandable in under 30 seconds
- Controls feel reliable
- Basic fail/win states work
- No known blocker bugs
- Investigation flow is coherent
- HUD / UI is readable
- Audio defaults are not broken

## Technical checks
- Fresh install/build works
- No console spam or critical runtime errors
- Basic performance is acceptable on target device
- Resolution/scaling behaves reasonably
- Input works as expected

## Deployment
- Repo pushed cleanly
- Build/deploy target chosen
- Public URL tested
- Another device/browser tested if relevant

## Release artifacts
- short game description
- controls list
- screenshots or GIFs
- known issues list
- next-improvements backlog
