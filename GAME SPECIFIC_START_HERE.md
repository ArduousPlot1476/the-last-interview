# Start Here

These files are the operating system for building this game with ChatGPT + Claude Code.

## What this pack is for
Use these files as uploaded project knowledge in a new chat so the next session starts with:
- the intended game direction,
- the workflow,
- the spec template,
- the Claude Code handoff protocol,
- playtest/debug structure,
- shipping standards.

## Intended game direction
The target is **not** the full “living city AI mystery sim” yet.

The correct build target is a **small vertical slice**:
- one location,
- one case,
- 3–5 NPCs,
- clue collection,
- contradiction / accusation / deduction loop,
- 2–3 possible outcomes,
- text-first presentation,
- optional lightweight memory/state,
- no open world,
- no creator mode,
- no large systemic simulation.

## How the next chat should begin
The next chat should:
1. confirm the smallest fun version,
2. finalize the spec,
3. lock the tech stack,
4. produce 3–5 milestones,
5. generate the first Claude Code handoff.

## Default principle
Do not jump straight into coding.
Plan first, keep scope tight, and make every milestone playable or directly testable.
